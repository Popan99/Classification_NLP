# Classification NLP: Классификация отзывов (Sentiment Analysis)

## Задача
Классификация отзывов на позитивные и негативные на основе анализа текстового содержания отзыва.

## Данные
- **Источник:** Отзывы пользователей с различных платформ
- **Размер:** 50,000 записей 
- **Признаки:**
  - `review` - текст отзыва
- **Целевая переменная:** `label` / `sentiment` (0 - негативный / 1 - позитивный)
- **Особенности данных:**
  - Не всегда структурированный текст на английском языке
  - Присутствие сленга, опечаток и специальных символов
  - Сбалансированный датасет
  - Различная длина текстов (от коротких до развёрнутых)

## Решение

### 1. EDA (Исследовательский анализ данных)
- Анализ структуры и качества данных
- Исследование распределения длины текстов отзывов
- Анализ баланса классов (позитивные vs негативные)
- Анализ часто встречающихся слов и n-грамм
- Выявление ключевых слов-индикаторов для каждого класса
- Визуализация облака слов (word clouds)

### 2. Baseline модели
- **CountVectorizer + классические ML алгоритмы:**
  - LogisticRegression
  - DecisionTreeClassifier  
  - RandomForestClassifier
  - LGBMClassifier
- **TF-IDF векторизация**
- **Очистка данных:** удаление стоп-слов, сохранение пунктуации, нормализация текста

### 3. Нейронные сети
- **RNN** - рекуррентная сеть для анализа последовательностей
- **CNN** - сверточная сеть для выявления локальных паттернов
- **LSTM** - долгосрочная память для сложных зависимостей
- **CNN + GloVe** - предобученные эмбеддинги слов

### 4. Предобработка
- Токенизация с помощью tiktoken
- Создание последовательностей для нейронных сетей
- Отбор важных токенов
- Стратифицированная кросс-валидация
